# AtomStack 海外客服 SOP 帮助中心

AtomStack 海外客服 SOP & 邮件模板帮助中心，涵盖购买咨询、退货退款、物流处理、工单创建等全流程操作规范。

## 在线访问

通过 GitHub Pages 访问：https://love520123333.github.io/atomstack-sop/

## 本地使用

直接双击 `sop_website.html` 即可在浏览器中打开。

## 项目文件

- `sop_website.html` - 生成的单页帮助中心
- `build_sop.py` - HTML 生成脚本
- `read_xlsx.py` - Excel 数据读取脚本
- `sheet_data.json` - 数据源
